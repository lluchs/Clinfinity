AttachTo

global func AttachTo(object to,
int callerVertex,
int targetVertex)

Attaches the calling object to the object to.  Note: This also changes the object order, so to is internally sorted before the calling object.  Furthermore, the target object may receive an attach event by providing a method called AttachEvent().

If the calling object isn’t already in an action with procedure ATTACH the function will set the action “Attach”.

When callerVertex or targetVertex are omitted, this function will add new vertices so that the object won’t move.  Important: Vertex indices are starting at 1 instead of 0.

Parameters

toObject to which the caller should be attached.
callerVertexThe caller object’s vertex which will be attached.
targetVertexThe vertex to attach to.

CopyVertices

global func CopyVertices(object from)

Copies all DefCore-defined vertices from another object.

The copied vertices are saved using an Effect, allowing easy removal using RemoveCopiedVertices.

Parameters

fromObject from which the vertices are copied.

StartCopiedVerticesRemoval

global func StartCopiedVerticesRemoval()

Improves performance of removing multiple vertices by only re-adding the remaining vertices in the end.

Important: You need to call EndCopiedVerticesRemoval after all calls to RemoveCopiedVertices are done.

RemoveCopiedVertices

global func RemoveCopiedVertices(object from)

Removes vertices which were previously copied using CopyVertices.

Important: This function will remove all user-defined vertices that aren’t copied from other objects.

Note: Use StartCopiedVerticesRemoval when calling this function repeatedly with different objects.

Parameters

fromObject from which the vertices were copied.

CopyChildrenVertices

global func CopyChildrenVertices(object child)

Copies all DefCore-defined vertices from all attached objects.  This is done recursively, thus the vertices of both directly and indirectly attached objects are copied.

Parameters

child[optional] The attached object where the copying starts.

RemoveCopiedChildrenVertices

global func RemoveCopiedChildrenVertices(object child,
bool rec)

Removes all vertices which were previously copied from all attached objects.  This is done recursively, all vertices of directly and indirectly attached objects are removed.
